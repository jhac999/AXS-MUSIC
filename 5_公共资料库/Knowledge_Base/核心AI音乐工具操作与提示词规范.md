# 核心AI音乐工具操作与提示词规范指南

本手册针对目前音乐制作全链路中的核心AI工具，提供底层逻辑研究、标准操作指南以及高阶提示词（Prompt）控制公式。

---

## 第一阶段：端到端生成平台 (Suno / Mureka / Udio)

这类平台的核心在于将“自然语言”转化为“声学特征”。熟练掌握元标签（Meta-tags）与风格堆叠是其核心技能。

### 1. 平台特性研究
*   **Suno V5**：擅长流行乐结构、拟真人声和歌词情感渲染。适合做主打单曲（Top-liner）。
*   **Mureka V8**：强项是宏观结构控制与和弦逻辑演绎，支持哼唱旋律输入。适合专业编曲打底。
*   **Udio**：声学高保真，极宽的频响范围。适合需要重低音轰炸的电音（EDM）、史诗配乐或嘻哈采样。

### 2. 标准提示词（Prompt）公式
在这些平台中，不建议写长篇大论的散文，而应使用**逗号分隔的标签矩阵**。
**黄金公式**：`[主风格] + [融合风格/副风格] + [情感/氛围] + [主导乐器] + [节奏/速度] + [人声特质]`

**高阶提示词示例**：
*   *流行电子（适合Suno）*：`Synth-pop, Retrowave, melancholic atmosphere, driving bassline, analog synthesizers, 120 BPM, ethereal female vocals, breathless.` (合成器流行，复古波，忧郁氛围，驱动贝斯线，模拟合成器，120BPM，空灵女声，充满呼吸感)
*   *重型配乐（适合Udio）*：`Cinematic epic, Dark Orchestral, Hans Zimmer style, brass ensemble, massive taiko drums, overwhelming tension, slow build up, no vocals.` (电影史诗，暗黑管弦乐，汉斯季默风格，铜管合奏，重型太鼓，压倒性张力，缓慢铺垫，无人声)

### 3. 结构元标签（Meta-tags）规范
将以下标签用中括号 `[]` 标注在歌词文本内，可以强制AI识别歌曲段落结构，防止歌曲结构失控。
*   `[Intro]`：前奏（通常为器乐）。
*   `[Verse 1]` / `[Verse 2]`：主歌，叙事铺垫，人声能量较低。
*   `[Pre-Chorus]`：预副歌，能量爬升。
*   `[Chorus]`：副歌，情感和器乐编排的最高潮。
*   `[Bridge]`：桥段，通常发生转调或节奏变化，打破重复感。
*   `[Guitar Solo]` / `[Beat Drop]`：强制插入一段器乐Solo或电音Drop。
*   `[Outro]` / `[Fade Out]`：尾奏，引导音乐淡出结束。

## 第二阶段：音频分离工具 (UVR5 / LALAL.AI / Moises)

### 1. 算法特性研究
*   **Ultimate Vocal Remover (UVR5)**：开源离线运算王者。通过本地GPU运算。支持MDX-Net与Demucs等多种模型。
*   **LALAL.AI**：云端商业API。基于频域掩膜技术，能够极快剥离人声与伴奏。
*   **Moises**：偏向现场乐手与教师使用，集成分轨、BPM与和弦检测一体。

### 2. UVR5 本地高阶使用方法
UVR5 能够实现比商业软件更干净的提取，但需要正确的模型搭配。
*   **提取主唱人声（去除和音与伴奏）**：
    *   模型选择：选择 `MDX-Net` 架构下的 `Kim_Vocal_2` 或 `MDX23C`。
    *   参数设置：勾选 `Vocal Only`，调整 `Aggression`（激进程度）至适中，过高会吞掉歌手的齿音，过低会残留镲片声。
*   **提取极其干净的伴奏（Instrumental）**：
    *   模型选择：选择 `VR Architecture` 中的 `5_HP-Karaoke-UVR`。
    *   参数设置：Window Size 调至 `320`，Aggression 适中。
*   **Ensemble（集成）模式**：当遇到极其拥挤的重金属或摇滚乐时，开启 Ensemble 模式，同时勾选 Demucs 与 MDX 模型让 AI 进行综合投票，可获得最高保真度。

## 第三阶段：AI虚拟人声合成 (ACE Studio 2.0)

### 1. 平台研究
不再使用切片拼接，而是使用神经网络基于MIDI输入实时生成具有高度呼吸感和共鸣的人声（Verse25引擎）。

### 2. 使用方法与调教规范
将ACE Studio作为VST插件挂载至DAW，或独立运行。
1.  **导入MIDI**：将编曲的旋律线MIDI导入，或利用音频转MIDI功能提取基调。
2.  **填词规范**：支持拼音与直接汉字输入。针对多音字，需手动双击音符使用拼音修正（如 `yue` / `le`）。
3.  **情绪曲线绘制（核心表现力）**：
    *   `Tension (张力)`：在副歌高潮处，用画笔工具拉高此参数，AI会模拟喉咙缩紧、发力嘶吼的声学状态。
    *   `Breathiness (气声)`：在主歌或抒情段落，提升此参数以增加人声的空气感和耳边呢喃的效果。
    *   `Pitch Bend (滑音与颤音)`：AI会自动生成颤音，若觉得不自然，可在参数栏降低默认颤音深度，手动绘制尾音的滑音（Portamento）。
4.  **Choir Mode（合唱模式）**：选中主旋律片段，右键点击“生成和声”，AI会基于你的主旋律自动配上三度或五度复调，瞬间生成人声墙（Vocal Wall）。

## 第四阶段：智能混音与母带 (iZotope Ozone 11 / Sonible)

### 1. 工具定位
*   **Sonible smart:EQ 4**：基于AI的自动频率均衡器，用于“轨道级”混音，解决各乐器打架的问题。
*   **iZotope Ozone 11**：工业级母带套件，用于“总线级”终极润色，决定发行的最终音质。

### 2. 标准混音工作流与设置
*   **消除频率掩蔽（使用 smart:EQ 4）**：
    1.  将插件分别挂在“人声（Vocal）”总线和“伴奏（Instrumental）”总线上。
    2.  将其加入同一个“组（Group）”。
    3.  拖拽人声图标至图层最上层（优先级最高）。
    4.  点击学习按钮（Learn），播放歌曲最热闹的副歌段落。AI会自动生成反向补偿曲线，压低伴奏中与人声冲突的频率（通常在中频），让人声立刻清晰浮现。
*   **智能母带一键匹配（使用 Ozone 11）**：
    1.  挂载在主输出轨道（Master Track）。
    2.  点击 `Master Assistant`（母带助手）。
    3.  **目标匹配（Target）**：上传一首你认为音质极佳的商业发行曲目。
    4.  播放你自己的副歌音频，AI将在一分钟内分析并配置：动态EQ（修正异常频段）、Imager（扩展立体声宽度）、Maximizer（提升整体 LUFS 响度至流媒体标准，如 -14 LUFS）。
    5.  微调：根据听感，手动调节 Assistant 面板中的 `Tone` (明亮/温暖) 与 `Width` (宽阔/紧凑) 滑块即可完成工业级母带制作。
