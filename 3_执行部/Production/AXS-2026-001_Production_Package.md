# 联合制作包 (Production Package)

- **订单号**: AXS-2026-001
- **生成节点**: 五大工作室并行输出结果汇总

## 1. 🎤 作词工作室 (由【黄伟文+方文山】主导)
**破题思路**：摒弃俗套的“飞驰、速度”，采用赛博朋克的冷色调名词与机器觉醒的概念。
```text
[Verse 1]
霓虹割破了夜的静脉 (Neon slices the night's veins)
液态金属在瞳孔里超载 (Liquid metal overloads the pupils)
不需要呼吸，不需要等待
只有 800 匹马力在动脉里澎湃

[Pre-Chorus]
心跳开始失真 (Heartbeat starts to distort)
地心引力失效的黄昏

[Chorus]
(爆点)
Let it burn! 撕裂这片电子的尘埃！
用光速冲破那道该死的屏障！
我是沉默的闪电，我是暴雨的主宰！
Let the engine roar, we leave the world behind!
```

## 2. 🎼 作曲工作室 (由【Hans Zimmer+Max Martin】主导)
**旋律弧线**：
- Verse 采用极简的半音阶下行（制造压抑的驾驶感）。
- Pre-Chorus 旋律线突然拔高一个八度，配合加速感。
- Chorus 采用最经典的流行大跳音程（洗脑 Hook），配合 Hans Zimmer 式的史诗级和声铺垫（i-VI-III-VII）。

## 3. 🎸 编曲工作室 (由【Calvin Harris】主导)
**配器蓝图**：
- [Verse 1]：绝对不放底鼓！只有厚重的 Reese Bass 和极其锋利的锯齿波（Saw Synth）琶音。
- [Pre-Chorus]：加入高频的合成器 Riser 和军鼓滚奏（Snare Roll），BPM 听觉翻倍。
- [Chorus]：全阵列开火！加入极其沉重的 808 底鼓，电影级铜管群组（Brass Section）在每小节第一拍砸下。

## 4. 🎛️ 混音工作室 (由【Jaycen Joshua】主导)
**声学参数**：
- **低频处理**：为了百万级车载音响，Kick（底鼓）的 50Hz 必须极度结实，Reese Bass 必须在 60Hz 以下做侧链避让（Sidechain），确保不轰头。
- **宽度**：合成器使用极其宽广的立体声（Haas Effect），人声死死锁在正中间（Mono）。

## 5. 🎚️ 母带工作室 (由【Sterling Sound】主导)
**交付标准**：
- **目标响度**：-7 LUFS（极致商业侵略性）。
- **特殊处理**：由于是车载电音，必须使用多段压缩狠狠压榨 100Hz-250Hz 的泥泞区，保持低频的“拳拳到肉”而不浑浊。
